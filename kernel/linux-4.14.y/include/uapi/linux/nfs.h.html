37/*
38 * NFS stats. The good thing with these values is that NFSv3 errors are
39 * a superset of NFSv2 errors (with the exception of NFSERR_WFLUSH which
40 * no-one uses anyway), so we can happily mix code as long as we make sure
41 * no NFSv3 errors are returned to NFSv2 clients.
42 * Error codes that have a `--' in the v2 column are not part of the
43 * standard, but seem to be widely used nevertheless.
44 */
