713 *
714 * @param p_hwfn
715 * @param rx_coal - Rx Coalesce value in micro seconds.
716 * @param tx_coal - TX Coalesce value in micro seconds.
717 * @param vf_id
718 * @param qid
719 *
720 * @return int
721 **/
